> This is a fork of [`NMaghfurUsman/kra-remote`](https://github.com/NMaghfurUsman/kra-remote), modified for use with [`opencomic-ai-training`](https://github.com/ollm/opencomic-ai-training).
> Unlike the original plugin, this version keeps the WebSocket connection permanently open. For this reason, it is not recommended to install it unless you explicitly need it.
>
> Requires Krita 5.3.0 or later.

## Installation

1. Open Krita.

2. Go to *Tools* -> *Scripts* -> *Import Python Plugin from Web*

3. A dialog will pop up. Copy this link into the input https://github.com/ollm/kra-remote/archive/refs/tags/v0.1.1.zip

   Press OK.

4. Restart Krita.

5. Enable the docker in *Settings* -> *Dockers* -> *Krita Remote*
